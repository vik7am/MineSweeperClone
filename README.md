# MineSweeper Clone
Play on Replit [Click Here](https://replit.com/@vikrantkataria/MineSweeperClone)
## Rules
1. The board is divided into tiles, with mines randomly distributed.
2. To win, you need to open all the tiles.
3. The number on a tile shows the number of mines adjacent to it. Using this information, you can determine tiles that are safe, and tiles that contain mines.
4. Interact, evolve and enjoy!

## How to Play
1. Player will Input the row and column of the tile which they want to open.
2. Player can place or remove flags on hidden tiles by adding a (-) sign to row and column values.
